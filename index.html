搭建blog

使用Hexo框架搭建了我的blog，hexo使用markdown解析文章，能方便地生成静态网页。

同时把博客托管在github pages上。 (for free)

0x01 所需环境

安装：

  • node.js
  • git

0x02 github建立仓库

  • github新建仓库，名为 username.github.io
  • 其中username一定是你的github用户名

0x03 安装hexo

  • 用npm包管理器安装Hexo
    npm install -g hexo

0x04 hexo操作

  • 建立自己的blog文件集 之后所有的操作都在此文件夹
  • hexo 初始化
    hexo init
  • hexo 生成
    hexo g
  • hexo 启动服务进行预览
    hexo s

0x05 将blog部署到github

  • 配置_config.yml 中的内容
  • (由于女拳)github将master改为了main,所以有些时间较早的教程可能branch填的是master,所以需要修改
  • 安装插件,一键上传代码
    npm install hexo-deployer-git --save
  • 将blog提交到github
    hexo d
    于是我们的blog就部署好啦,是不是非常简单高效？
